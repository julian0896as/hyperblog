git init = crea un repositorio local en el directorio actual.

git add = añade al staging los cambios realizados. se puede añadir "add ." para especificar que queremos que
se guarden los cambios en todo el directorio actual

git commit -m "dblkjher" = se suben los cambios con un mensaje de identificación al repositorio

git log = muestra el historial de commits con su respectivo autor.

git status = muestra el estado de los archivos untraked and recent traked (staging)

git branch "nombre" = crea una rama para trabajar de manera independiente al master. 

git checkout = me envía a la rama donde quiero trabajar

git merge = fuciona las ramas ej: eb master git merge cabecera // esto pasaría los cambios de cabecera a master
